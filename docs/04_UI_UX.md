# UI/UX Design

## 1. Visual Identity

### 1.1. Theme: "Corporate Noir"

A visual style that blends the mundane corporate world with detective noir tension.

| Element            | Description                                                                   |
| ------------------ | ----------------------------------------------------------------------------- |
| **Overall mood**   | Late nights in the office, coffee-stained reports, monitor glow in a dim room |
| **Color approach** | Muted, professional, with strategic accent highlights                         |
| **Typography**     | Clean, readable, professional sans-serif                                      |
| **Textures**       | Subtle paper grain, folder aesthetics, document shadows                       |
| **Motion**         | Subtle, purposeful â€” nothing flashy                                           |

## 5. Animations & Transitions

### 5.1. Animation Principles

| Principle      | Implementation                            |
| -------------- | ----------------------------------------- |
| **Purposeful** | Animations convey meaning, not decoration |
| **Fast**       | 150-300ms for most transitions            |
| **Subtle**     | Don't distract from investigation         |
| **Reducible**  | Respect `prefers-reduced-motion`          |

### 5.2. Key Animations

```css
/* ========================================
   TRANSITIONS
   ======================================== */

/* Page transitions */
.page-enter {
  opacity: 0;
  transform: translateY(10px);
```

## 7. Accessibility

### 7.1. WCAG 2.1 AA Compliance

| Requirement             | Implementation                                  |
| ----------------------- | ----------------------------------------------- |
| **Color contrast**      | 4.5:1 minimum for text, 3:1 for large text      |
| **Focus indicators**    | Visible 2px outline on all interactive elements |
| **Keyboard navigation** | Full app navigable without mouse                |
| **Screen reader**       | ARIA labels, live regions for dynamic content   |
| **Reduced motion**      | Respect `prefers-reduced-motion`                |
| **Text scaling**        | UI works at 200% zoom                           |

### 7.2. Keyboard Shortcuts

| Shortcut           | Action                       |
| ------------------ | ---------------------------- |
| `j` / `k`          | Navigate inbox up/down       |
| `o`                | Open selected document       |
| `p`                | Pin current selection        |
| `/`                | Focus search                 |
| `c`                | Toggle chat panel            |
| `b`                | Toggle board                 |
| `h`                | Request hint                 |
| `?`                | Show keyboard shortcuts      |
| `Esc`              | Close modal/panel            |
| `Cmd/Ctrl + Enter` | Submit (when in submit form) |
